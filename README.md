# Smart Wardrobe 👗

A application that helps users manage their wardrobe and plan outfits based on weather conditions.

## Features
- Add, update, and delete clothing items
- Create and manage outfits
- Filter clothes by weather type
- User login system

## Technologies Used
- Java (Swing)
- MySQL
- JDBC

## Database
Import `smartwardrobe.sql` to set up the database before running the app.
